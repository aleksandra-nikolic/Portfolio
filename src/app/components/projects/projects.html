Patients List

A simple patient management application built with Angular. It allows users to add new patients with their basic information (name, age, diagnosis), edit or delete existing records, and search through the patient list by any field. The app includes empty states for both an empty list and no search results. All data is stored in localStorage.

02

Pet Name Generator

A fun and interactive app that helps users find the perfect name for their pet. The user starts by selecting their pet type, then moves to a filters page where they choose the gender and name style. Based on those filters, the app generates a name from a manually created database. Users can save their favorite names, which are stored in localStorage and accessible via a favorites modal on the home page — with an empty state if no names are saved yet. Each page is connected through Angular routing.

Angular Typescript CSS HTML Routes Local Storage
GitHub Live demo
03

Movies App

An app for browsing movies and TV shows. The home page displays trending content, while separate pages list all available movies and TV shows — each with a search option by name. Content is displayed as cards with a poster image and title, similar to most movie browsing sites. Clicking on any title opens a details page with more information. Data is fetched from the TMDB API. Empty states are included for when search results don't match.
