# Portfolio
My personal portfolio website built with Angular. It showcases three projects I built while learning frontend development.
## Projects

-Patients List — A patient management app with CRUD operations, search functionality, and localStorage.
=Pet Name Generator — An interactive app that generates pet names based on selected filters, with a favorites list stored in localStorage.
=Movies032 — A movie and TV show browsing app that fetches data from the TMDB API.

## Built With

-Angular
-TypeScript
-HTML & CSS
-Tailwind CSS
